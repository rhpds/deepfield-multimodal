== Technology

* *Intel Xeon 6* — CPU inference for deterministic nanoagents and rule-backed microagents. No GPU required for runtime classification.
* *Intel Gaudi 3* — Optional GPU acceleration for macroagent LLM reasoning when configured.
* *Intel TDX* — Trust Domain Extensions for secure bootstrap analysis. Data encrypted in-use.
* *Red Hat OpenShift* — Container platform with OAuth proxy, cluster-reader access for live monitoring.
* *Qwen 3 235B on Intel* — Open-weight frontier model for one-time bootstrap semantic analysis.
* *Granite 3.2 8B on Intel Xeon* — CPU-optimized model for runtime micro/macro classification.
